# WebVilo — Development Guide

WebVilo ist unsere professionelle Website. Dieses Dokument definiert unsere Standards.

## 🎯 Brand & Vision
- Moderne, professionelle Präsentation
- Mobile-first Design
- Schnell & performant
- Accessibility (A11y) Standard

## 🛠 Tech Stack
- **Framework:** React 18+ mit Hooks
- **Styling:** TailwindCSS
- **Build:** Vite
- **Language:** TypeScript
- **Package Manager:** npm

## 📁 Projekt-Struktur


src/
├── components/        # Wiederverwendbare Komponenten
│   ├── Header.tsx
│   ├── Hero.tsx
│   ├── Features.tsx
│   └── Footer.tsx
├── pages/            # Seiten
├── styles/           # Global CSS
├── App.tsx
└── main.tsx
## ✅ Code Standards
- Funktionale Komponenten
- Hooks für State Management
- TypeScript überall
- Aussagekräftige Namen

## 🎨 Design System
- Modern & Clean Design
- Mobile-first Responsive
- Accessibility Standard

## 📝 Commits
- `feat:` Neue Features
- `fix:` Bug Fixes
- `docs:` Dokumentation

## 🔄 Workflow
1. Branch: git checkout -b feature/name
2. Code + Test lokal
3. Push: git push origin feature/name
4. Pull Request
5. Review + Merge

## 🚀 Deployment
- Keine Console-Errors
- Responsive auf allen Screens
- Performance (Lighthouse 90+)
